# GPS / IMU Track Recorder Wave 2

## 第 2 波內容
- PWA 安裝與 service worker
- App / Track 分享按鈕
- 自動備份槽（localStorage）
- 可下載自動備份 JSON
- modal 標記管理
- GitHub Pages 可直接部署

## 部署
1. 解壓縮後，把全部檔案上傳到 repo 根目錄
2. 確認主檔是 `index.html`
3. GitHub Pages 設成 `main` + `/ (root)`

## 注意
- 離線可開 App Shell
- 地圖瓦片仍需網路
- 真正離線地圖需改用預載瓦片或 MBTiles 方案
